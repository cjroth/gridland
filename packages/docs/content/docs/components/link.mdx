terminal UI.

## Installation

<Tabs items={['bun', 'npm', 'yarn', 'pnpm']}>
<Tab value="bun">
```bash title="Terminal"
bun add @gridland/ui
```
</Tab>
<Tab value="npm">
```bash title="Terminal"
npm install @gridland/ui
```
</Tab>
<Tab value="yarn">
```bash title="Terminal"
yarn add @gridland/ui
```
</Tab>
<Tab value="pnpm">
```bash title="Terminal"
pnpm add @gridland/ui
```
</Tab>
</Tabs>

## Usage

```tsx title="Example"
import { Link } from "@gridland/ui"

<Link url="https://opentui.com">Visit opentui.com</Link>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Link text content |
| `url` | `string` | Target URL |
| `underline` | `"solid" \| "dashed" \| "dotted" \| "none"` | Underline style (default `"solid"`) |
