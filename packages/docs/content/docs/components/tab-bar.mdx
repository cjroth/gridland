The `TabBar` component from `@gridland/ui` renders a horizontal, single-line selector
that displays a row of text options with one visually marked as selected.

## Usage

```tsx title="Example"
import { TabBar } from "@gridland/ui"

<TabBar label="View" options={["Files", "Search", "Git"]} selectedIndex={0} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `undefined` | Text label shown before the options |
| `options` | `string[]` | -- | Array of option strings to display |
| `selectedIndex` | `number` | -- | Zero-based index of the currently selected option |
| `focused` | `boolean` | `true` | Whether the tab bar appears focused |
| `activeColor` | `string` | `"cyan"` | Foreground color of the selected option when focused |
