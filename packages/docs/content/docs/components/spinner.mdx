with customizable text and color.

## Usage

```tsx title="Example"
import { Spinner } from "@gridland/ui"

<Spinner text="Loading..." color="cyan" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | `"Loading"` | Text displayed next to the spinner |
| `color` | `string` | `"gray"` | Color of the spinner and text |
| `interval` | `number` | `100` | Animation frame interval in milliseconds |
